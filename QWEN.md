# Qwen CLI Context File

## Project Overview
This is the **karobar-sahulat-web-pos** project, a web-based Point of Sale (POS) system designed to help businesses manage their sales operations efficiently.

## Project Structure
The main project directories are:
- `karobar-sahulat-backend` - Backend server and API
- `karobar-sahulat-mobile-app` - Mobile application
- `karobar-sahulat-web-pos` - Web-based Point of Sale system (this project)

## Purpose of This File
This QWEN.md file serves as context for Qwen CLI to understand that when working in this directory, we're focusing on the web-based POS system component of the Karobar Sahulat project.

## Scope
When working on this project, focus on:
- Web-based POS functionality
- Frontend development for the POS system
- Integration with the backend API
- User interface for point of sale operations
- Sales management features
- Inventory tracking through the web interface
- Customer management via web POS

## Context Clarification
If asked to work on "WEB POS" or "karobar-sahulat-web-pos", understand this refers to the web-based Point of Sale system component of the larger Karobar Sahulat business management suite.
